Here are the general guidelines on how to use the program
Step 1: change the current directory of your MATLAB to this folder
Step 2: run shaft_design from the command window
Step 3: run the function Load_with_position by calling it in the command window
after running the shaft_design.m and don't forget to insert the loads and their position 
                  Example:   Loads_with_position([0.4 pulley],[0.8 gear],[1 sprocket],[1.3 pulley])
Step 4: follow the instructions that program gives you
